# Integrative Wellness Partners

Concierge medicine website for Dr. Samar Riaz, NMD.

## Tech Stack

- **React** 18.2
- **Vite** 5.0
- **React Router** 6.20
- **Tailwind CSS** 3.3

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Layouts

The site includes three different design layouts:

- **Layout 1**: Refined minimalist with elegant typography
- **Layout 2**: Bold modern luxury with dramatic gradients
- **Layout 3**: Editorial magazine-inspired (recommended)

Visit `/layout1`, `/layout2`, or `/layout3` to preview each design.

## Deployment

Configured for Netlify deployment:

- Build command: `npm run build`
- Publish directory: `dist`
- Node version: 18
